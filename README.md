## Installation

Download and build project

```sh
git clone git@github.com:gt-ostis-dev/gt.ostis.git
cd gt.ostis/scripts 
./prepare.sh     

```

Start project

```sh
./restart_sctp.sh 

```

Start web-server

```sh
./run_scweb.sh   

```

Open in browser <a href="http://localhost:8000/">http://localhost:8000/</a>
